What's next

This view currently shows project count — the number of distinct projects at each goal × department intersection. It's an honest signal about where the team's work is spread, but it counts a small effort the same as a large one.

When the tracker exposes its time_entries data via AGOL, this page will switch to showing actual time invested — hours logged per goal × department per quarter. At that point we'll likely add a quarter dimension so leadership can see how investment has shifted over time. The page name and URL will stay the same; the underlying signal will get sharper.

Until then, count is the most accurate framing — and a useful map of where the program reaches today.
